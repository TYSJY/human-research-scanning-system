# Open Source Audit

这轮整理重点不只是“清理仓库”，而是把项目往 **专业科研助手** 的定位上推了一步。

## 已完成

- 清理 `.pytest_cache`、`__pycache__`、预览 HTML、旧 preview workspace、旧 snapshot demo
- 把历史交付、迁移说明、重构评审从仓库顶层收进 `docs/archive/`
- 增加 `examples/`，让首次访问者能直接看到成果物导向的示例包
- 增加 `ros showcase`，把工作区导出成面向研究协作的成果物包
- 增加 package data bundling，让安装版也能拿到模板、控制平面与演示项目
- 补齐 GitHub 社区文件与 CI
