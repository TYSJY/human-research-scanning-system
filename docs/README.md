# Docs

这份仓库不只是一个本地工作台原型，而是一个面向专业研究工作的 **AI 科研助手系统**。

为了让首次访问者快速看懂仓库，文档按四个层次组织：

## 先看这些

- `../README.md`：项目定位、四大核心能力、快速开始
- `product_positioning.md`：为什么它不是普通聊天工具，而是专业科研助手
- `scenarios.md`：三类高价值科研场景
- `deliverables.md`：这个系统真正交付什么成果物
- `agent_roles.md`：控制平面的研究角色分工
- `evaluation_framework.md`：如何评估“专业科研助手”是否靠谱

## 目录说明

- `reference/`：当前仍有参考价值的架构说明与技术设计
- `maintainers/`：维护者文档、开源审计与仓库约定
- `archive/`：历史交付文档、迁移说明与旧评审材料
- `assets/README.md`：README 展示图与社交分享图资产
