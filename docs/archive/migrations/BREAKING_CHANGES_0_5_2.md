# Breaking Changes 0.5.2

本次没有引入项目数据层 breaking change。

## 非数据层变化

### 1. UI 视觉与布局明显变化
- 深色控制台风格改为浅色工作台风格
- 某些低频操作不再默认展开，而是收进折叠区

### 2. D 线管理区改成锚点导航 + 折叠管理块
这会改变之前的视觉位置，但功能入口仍然保留。

### 3. Python 包名修正
- `pyproject.toml` 中的包名从历史遗留的 `research-os-v4-3` 修正为 `research-os`
- 版本号更新为 `0.5.2`

## 兼容性说明

- CLI 命令 `ros` 保持不变
- 项目目录结构保持兼容
- 现有 studio 状态文件无需手工迁移
