# Product Positioning

## 一句话定位

**Research OS 是一个面向专业研究工作的 AI 科研助手，帮助研究者完成文献综述、研究设计、学术写作与可复现研究管理。**

## 这不是普通聊天工具

普通 AI 聊天工具更擅长一次性问答；Research OS 更强调以下四件事：

1. **证据链**：每条主张都能追溯到 evidence、run、artifact。
2. **研究结构**：问题、claim、MVP、result、deliverable 都有明确状态。
3. **成果物导向**：目标不是“聊得不错”，而是交付 review brief、evidence matrix、experiment plan、paper outline。
4. **复现与审计**：状态、日志、任务与结果可以持续复查。

## 四大核心能力

### 1. Literature Review
- 研究问题拆解
- 检索策略与关键词
- evidence registry
- gap analysis

### 2. Study Design
- 研究主张与 acceptance checks
- MVP 设计
- 实验计划
- 风险与 reviewer objections

### 3. Scholarly Writing
- title / abstract
- outline
- contribution constraints
- rebuttal notes

### 4. Reproducible Research Ops
- run registry
- result provenance
- audit report
- exportable showcase package
