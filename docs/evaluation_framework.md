# Evaluation Framework

这个项目要成为“专业科研助手”，必须可评估，而不是只展示界面。

## 当前已经覆盖的检查方向

- metrics presence
- result value presence
- artifact integrity
- claim / result consistency
- evidence coverage
- provenance completeness

## 建议继续补强的评测

### Literature Review 质量
- evidence 是否完整
- 主题归纳是否覆盖主要分支
- gap 分析是否保守

### Research Design 质量
- claim 是否可证伪
- acceptance checks 是否明确
- MVP 是否足够小且可运行

### Scholarly Writing 质量
- 是否只引用已注册结果
- 摘要与结论是否与证据一致
- 是否保留不确定性表述

### Reproducibility 质量
- run provenance 是否可追踪
- 输出文件是否齐全
- 人工 gate 是否完整
