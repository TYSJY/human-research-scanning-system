# Deliverables

一个“专业科研助手”必须把结果沉淀成成果物，而不是停留在聊天记录里。

## 默认成果物

- `research_brief.md`：研究项目摘要、当前证据、主张、结果与下一步
- `evidence_matrix.csv`：证据矩阵，便于审阅与外部协作
- `deliverable_index.md`：当前主输出、run 结果、关键文件索引
- `runtime_audit_report.md`：运行与审计状态
- `notes/*.md`：阶段性研究笔记与写作骨架

## 为什么成果物重要

- 方便与导师、合作者、审稿人沟通
- 便于回顾“哪些结论有证据、哪些还只是草稿”
- 让项目离开单次对话后仍可延续
