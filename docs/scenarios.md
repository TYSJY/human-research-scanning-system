# High-Value Scenarios

下面三类场景最能体现“专业科研助手”的价值，而不是普通聊天机器人。

## 场景 1：从研究问题到文献综述包

输入：
- 一个研究主题
- 基础约束（时间、数据库、目标会议）

输出：
- 搜索策略
- evidence registry
- review brief
- gap analysis
- evidence matrix

## 场景 2：从已有证据到研究设计

输入：
- 已有 evidence
- 一个希望验证的主张
- 预算与数据约束

输出：
- claim graph
- acceptance checks
- MVP
- experiment plan
- risk checklist

## 场景 3：从结果到论文写作骨架

输入：
- 结构化 result registry
- 已采纳的 primary artifacts
- 目标投稿方向

输出：
- title / abstract
- outline
- related work brief
- rebuttal notes
- deliverable index
