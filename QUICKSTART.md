# Quickstart

## 1. 安装

```bash
python -m pip install -e .
```

## 2. 直接打开官方 demo

```bash
ros quickstart --launch-ui
```

## 3. 让系统继续推进一步

```bash
ros run <项目路径>
```

## 4. 导出研究成果物

```bash
ros showcase <项目路径>
```

这会额外生成：
- `reports/research_brief.md`
- `reports/evidence_matrix.csv`
- `reports/deliverable_index.md`

## 5. 做一次健康检查

```bash
ros audit <项目路径>
ros doctor <项目路径>
```
