# Examples

这里放的是 **成果物包**，不是运行时工作区。

目标是让第一次访问仓库的人，一眼看到这个项目能交付什么，而不是先去理解底层状态机。

## 当前示例

- `literature_review_pack/`：研究问题 -> 搜索策略 -> 证据矩阵 -> gap analysis
- `study_design_pack/`：已有证据 -> claim -> acceptance checks -> experiment plan
- `writing_pack/`：已注册结果 -> abstract / outline / reviewer response skeleton
